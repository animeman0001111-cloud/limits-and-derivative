TECHNIQUES OF DIFFERENTIATION
Basics

Constant & Power

d/dx(c)=0
d/dx(xn) = n xn−1
Log

Logarithmic

d/dx[ln(u)] = u'/u
d/dx[log a u] = u'/(u ln a)
Exp

Exponential

d/dx(eu) = eu·u'
d/dx(au) = auln(a)·u'
Trig

Trigonometric

sin x → cos x | cos x → −sin x
tan x → sec²x | cot x → −csc²x
sec x → sec x tan x
csc x → −csc x cot x
Rule

Quotient Rule

(u/v)' = (v u' − u v') / v²
Rule

Product Rule

(u·v)' = u v' + v u'
